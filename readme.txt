Σημειώσεις υλοποίησης

Desktop:	Στο Search Page δεν έχουμε υλοποιήσει τη λειτουργία του να εισάγει ο χρήστης κάποιο
			search term και αυτό να παραμένει στο search box, καθώς αυτό απ' ότι ψάξαμε χρειάζεται
			υλοποίηση server κάτι που λόγω έλειψης χρόνου δεν μπορέσαμε να υλοποιήσουμε. Αντ'αυτού,
			υλοποιήσαμε το Search Page στατικά. Επίσης για λόγους απλότητας υλοποιήσαμε 3 επιλογές 
			για κάθε κατηγορία φίλτρων εκτός από την επιλογή του rating, όπου υλοποιήσαμε 5 επιλογές.
			Ακόμα, δεν παρέχουμε τη λειτουργικότητα του συνδυασμού περισσότερης από μία κατηγορίας φίλτρων.
			Ο χρήστης μπορεί να επιλέγει μόνο μια κατηγορία φίλτρων κάθε φορά και να βλέπει τα αποτελέσματα
			να αλλάζουν βάσει της επιλογής των φίλτρων που κάνει.			
			Στο Product Page αντικαταστήσαμε το δεύτερο slideshow με μία ενιαία εικόνα του καλλιτέχνη,
			καθώς αντιμετωπίσαμε πρόβλημα στο να έχουμε δύο διαφορετικά slideshows στο ίδιο page χωρίς
			να υπάρχει collision στη λειτουργία τους.

Mobile:		Ως σημείο αναφοράς για την οθόνη του κινητού χρησιμοποιήσαμε το iPhone 8 Plus, το οποίο έχει
			μεγαλύτερη οθόνη από τα συμβατικά κινητά. Βάσει της οθόνης αυτής υλοποιήσαμε όλα τα Mobile pages.
